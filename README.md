# IntPass

> 선린인터넷고등학교 모바일 학생증 서비스

## Setup
This project uses yarn berry, so please use yarn
```
yarn install
```

## Development Server
Start the development server on http://localhost:8080
```
yarn dev
```

## Production
Build the application for production:
```
yarn build
```

## Environment
```
PUBLIC_API_URL
```
